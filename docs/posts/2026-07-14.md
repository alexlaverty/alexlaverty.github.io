# Site rename, GoatCounter analytics, and this daily blog

Spent the morning doing housekeeping on this site with Claude Code. The
site now goes by Laverty.io to match the custom domain, with my name kept
in the footer and page metadata. Added GoatCounter so I can finally see
if anyone reads these pages.

The bigger change is what you're reading: a daily log. Posts land on the
home page like a normal blog, and a small `new-daily.py` script scaffolds
each day's entry so there's no excuse not to write one. The idea is short
notes on what I did each day, with anything worth keeping written up
properly in the topic sections.
